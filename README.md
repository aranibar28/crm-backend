# Sistema CRM/ERP (Backend)

## Requisitos:
- vscode
- git
- node 14.x
- mongoDB
- cuenta gmail
- cuenta cloudinary

## Variables de Entorno

```
PORT=3000
MONGO_DB=mongodb+srv://<user>:<password>@cluster0.o7ft6.mongodb.net/<namedb>
SECRET_KEY=password
EMAIL=example@gmail.com
PASSWORD=xxxxxx
CLOUD_NAME=xxxxxx
CLOUD_API_KEY=xxxxxx
CLOUD_API_SECRET=xxxxxx
```

## Run application
```
npm install
```

```
npm start
```
